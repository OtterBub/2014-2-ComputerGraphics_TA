` ( 숫자키 1 옆 키 ) - Console창을 활성화 시킵니다. ( 투명 회색 )
--------- Console Command ---------------
"r" 또는 "reset" - 메인 씬을 재 실행 시킵니다.

Click and Drag - 슬라이스를 하기위한 선을 긋습니다.

a / d - 컨트롤 가능 도형을 X축으로 움직입니다.
w / s - 컨트롤 가능 도형을 Y축으로 움직입니다.

r / R - 컨트롤 가능 도형을 0.05배씩 음/양 으로 스케일 합니다. ( ClippingTest 씬에서는 작동 안합니다. )

메인 메뉴 실행시
---------- EasyVer ---------
교수님께서 처음에 말씀하신 A 선분 -> C 선분 클릭
나뉘어진 도형중 왼쪽이 선택된 도형,
B선분 -> D선분 클릭
나뉘어진 도형중 아랫 부분이 선택 도형

---------- RealSliceVer ---------
처음에 보여드렸던 씬입니다.

---------- ClippingTest ---------
두 네모가 겹쳐서 클리핑 되는것을 테스트 한 씬입니다.
보여주면 좋을것 같아 포함합니다.

오른쪽 사각형은 역시 리얼 슬라이스 버전으로 슬라이싱 됩니다.

각 씬의 오른쪽 아래 MainMenu 버튼을 누르면 언제든지 뒤로 돌아가기가 가능합니다.



****넓이쪽에서 스케일 된 후의 면적 구하는 것이 잘 안되어서 빼고 진행하였습니다.
